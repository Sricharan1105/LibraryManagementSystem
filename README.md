# 📚 Library Management System

A console-based Library Management System built with **Core Java**, demonstrating Object-Oriented Programming principles including Classes, Objects, Inheritance, Collections, and Exception Handling.

---

## 🚀 Features
- ➕ Add new books and register members
- 📖 Borrow and return books
- 🔍 Search books by title or author
- 📋 View all books and members
- ✅ Availability tracking for each book
- 👤 Member-wise borrowed book tracking

---

## 🛠️ Tech Stack
- Java (Core Java)
- OOP Concepts
- Java Collections (ArrayList, List)
- Java Streams API

---

## 📂 Project Structure
```
LibraryManagementSystem/
│
├── src/
│   ├── Main.java       # Entry point
│   ├── Library.java    # Core library logic
│   ├── Book.java       # Book entity class
│   └── Member.java     # Member entity class
└── README.md
```

---

## ⚙️ How to Run

### Compile
```bash
cd src
javac Main.java Book.java Member.java Library.java
```

### Run
```bash
java Main
```

---

## 🎯 OOP Concepts Used
- **Encapsulation** – Private fields with getters/setters in Book and Member classes
- **Abstraction** – Library class abstracts all operations
- **Collections** – ArrayList used to manage books and members
- **Streams** – Java Streams used for searching and filtering
